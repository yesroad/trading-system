# Dow Theory (다우 이론)

Charles Dow가 개발한 시장 분석의 기초 이론

## 6가지 기본 원칙

### 1. 시장은 모든 것을 반영한다

**"The Market Discounts Everything"**

- 가격에는 모든 정보가 반영됨
- 뉴스, 실적, 기대, 심리 모두 포함
- 펀더멘털 분석 불필요 (기술적 분석만으로 충분)

### 2. 시장에는 3가지 추세가 있다

#### Primary Trend (주추세): 1년 이상
- 강세장 (Bull Market): 지속적인 상승
- 약세장 (Bear Market): 지속적인 하락

#### Secondary Trend (2차 추세): 3주~3개월
- 주추세에 반대되는 조정
- 주추세의 33~66% 되돌림

#### Minor Trend (단기 추세): 3주 이하
- 단기 변동성
- 노이즈로 간주

```
    Primary Trend (상승)
         /\    /\    /\
        /  \  /  \  /  \  ← Secondary (조정)
       /    \/    \/    \
      /                  \
     /   Minor (단기)     \
```

### 3. 주추세는 3단계로 구성된다

#### 강세장 3단계

**1단계: 축적 (Accumulation)**
- 저가 매수 (스마트 머니)
- 일반 투자자는 비관적
- 거래량 낮음

**2단계: 대중 참여 (Public Participation)**
- 가격 상승 가속화
- 일반 투자자 진입
- 거래량 증가

**3단계: 과열 (Excess)**
- 모든 사람이 낙관적
- 뉴스도 긍정적
- 스마트 머니 매도 시작

#### 약세장 3단계

**1단계: 분산 (Distribution)**
- 스마트 머니 매도
- 일반 투자자는 여전히 낙관적

**2단계: 공황 (Panic)**
- 급격한 하락
- 일반 투자자 손절매

**3단계: 절망 (Despair)**
- 모든 사람이 비관적
- 저가 형성

### 4. 평균은 서로 확인해야 한다

**"Averages Must Confirm Each Other"**

Charles Dow는 2개 지수 사용:
- Dow Jones Industrial Average (산업)
- Dow Jones Transportation Average (운송)

**신호:**
- 두 지수 모두 신고가 → 강세장 확인
- 한 지수만 신고가 → 다이버전스 (경고)

**현대 적용:**
- S&P 500 + NASDAQ
- 대형주 + 소형주
- 주식 + 채권

### 5. 거래량은 추세를 확인한다

**"Volume Must Confirm the Trend"**

#### 건강한 상승 추세
- 상승 시: 거래량 증가
- 하락 시: 거래량 감소

#### 건강한 하락 추세
- 하락 시: 거래량 증가
- 반등 시: 거래량 감소

#### 경고 신호
- 신고가 but 거래량 감소 → 추세 약화
- 신저가 but 거래량 낮음 → 하락 세력 약화

### 6. 추세는 명확한 반전 신호가 나올 때까지 지속된다

**"A Trend Remains in Effect Until a Clear Reversal"**

#### 상승 추세
- 지속적인 고점/저점 상향
- Higher Highs, Higher Lows

```
    HH
   /  \    HH
  /    \  /
 /  HL  \/
/      HL
```

#### 하락 추세
- 지속적인 고점/저점 하향
- Lower Highs, Lower Lows

```
\      LH
 \    /  \
  \  /    \  LH
   \/  LL  \
        LL  \
```

#### 추세 반전 신호
- 상승 추세: Lower Low 발생
- 하락 추세: Higher High 발생

## 실전 적용

### 추세 확인 체크리스트

**강세장 확인:**
1. ✓ 가격이 Higher Highs, Higher Lows
2. ✓ 두 주요 지수 모두 상승
3. ✓ 상승 시 거래량 증가
4. ✓ 2차 조정이 주추세의 33-66% 이내

**약세장 확인:**
1. ✓ 가격이 Lower Highs, Lower Lows
2. ✓ 두 주요 지수 모두 하락
3. ✓ 하락 시 거래량 증가
4. ✓ 반등이 제한적

### 매수 타이밍

- 주추세 상승 확인
- 2차 조정(33-50% 되돌림) 완료
- 거래량 확인
- 다른 지수도 확인

### 매도 타이밍

- 추세 반전 신호 (Lower Low)
- 거래량 다이버전스
- 다른 지수 확인 실패

## 현대적 수정

### 3단계 → 더 세분화
- 축적 → 마크업 → 분산 → 마크다운

### 2개 지수 → 다중 확인
- 섹터 로테이션
- Breadth 지표
- 국제 시장

### 거래량 → 다양한 지표
- On-Balance Volume (OBV)
- Accumulation/Distribution
- Money Flow Index (MFI)

## 한계

- **지연 지표**: 추세 확인까지 시간 소요
- **주관성**: "명확한 반전"이 모호할 수 있음
- **변동성 무시**: 단기 변동성 고려 안 함
- **레인징 시장**: 횡보장에서 무용

## 다우 이론 + 다른 기법

- **Dow + Elliott Wave**: 3단계 = 5-3파동
- **Dow + Fibonacci**: 2차 조정 = 38-61.8% 되돌림
- **Dow + Moving Average**: 추세 확인 (50/200 MA)
