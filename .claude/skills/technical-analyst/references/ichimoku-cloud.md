# Ichimoku Cloud (일목균형표)

일본 저널리스트 Goichi Hosoda가 개발한 종합 기술적 분석 시스템

## 5가지 구성 요소

### 1. 전환선 (Tenkan-sen, 転換線)
```
전환선 = (9일 최고가 + 9일 최저가) / 2
```
**의미:** 단기 추세 (9일 중간값)
**색상:** 파란색 (보통)

### 2. 기준선 (Kijun-sen, 基準線)
```
기준선 = (26일 최고가 + 26일 최저가) / 2
```
**의미:** 중기 추세 (26일 중간값)
**색상:** 빨간색 (보통)

### 3. 선행스팬 A (Senkou Span A, 先行スパン A)
```
선행스팬 A = (전환선 + 기준선) / 2
※ 26일 앞에 표시
```
**의미:** 구름 상단 또는 하단

### 4. 선행스팬 B (Senkou Span B, 先行スパン B)
```
선행스팬 B = (52일 최고가 + 52일 최저가) / 2
※ 26일 앞에 표시
```
**의미:** 구름 상단 또는 하단

### 5. 후행스팬 (Chikou Span, 遅行スパン)
```
후행스팬 = 현재 종가
※ 26일 뒤에 표시
```
**의미:** 현재 가격의 과거 추세와의 관계

## 구름 (Kumo, 雲)

**선행스팬 A와 B 사이 영역**

```
가격
  ↑
  │     /----구름(저항)----\
  │    /                    \
  │   /                      \
  │  /                        \
  │ /                          \
  └──────────────────────────────→ 시간
```

### 구름 색상
- **녹색 구름**: 선행스팬 A > 선행스팬 B (상승 추세)
- **빨간 구름**: 선행스팬 A < 선행스팬 B (하락 추세)

### 구름 두께
- **두꺼운 구름**: 강한 지지/저항
- **얇은 구름**: 약한 지지/저항

## 매수 신호

### 강한 매수 (모든 조건 충족)
1. **가격 > 구름** (상승 추세)
2. **전환선 > 기준선** (골든크로스)
3. **후행스팬 > 26일 전 가격** (모멘텀 확인)
4. **녹색 구름** (추세 일치)

### 중간 매수
- 가격이 구름 아래 → 구름 위로 돌파 (구름 브레이크아웃)
- 전환선이 기준선 상향 돌파

### 약한 매수
- 가격이 구름 내부에서 반등

## 매도 신호

### 강한 매도 (모든 조건 충족)
1. **가격 < 구름** (하락 추세)
2. **전환선 < 기준선** (데드크로스)
3. **후행스팬 < 26일 전 가격**
4. **빨간 구름**

## 지지/저항 레벨

**우선순위:**
1. 구름 (가장 강력)
2. 기준선
3. 전환선

**두꺼운 구름 돌파 = 강한 추세 전환 신호**

## 실전 적용

### 추세 확인
```
강세장: 가격 > 전환선 > 기준선 > 구름
약세장: 가격 < 전환선 < 기준선 < 구름
```

### 진입 타이밍
- 구름 돌파 시 진입
- 전환선/기준선 크로스 확인 후

### 손절매 설정
- 강세장: 구름 하단
- 약세장: 구름 상단
- 또는 기준선

### 목표가
- 다음 구름까지
- 또는 두꺼운 구름 직전

## 시간대별 설정

### 기본 설정 (일봉)
- 전환선: 9일
- 기준선: 26일
- 선행스팬 B: 52일

### 단기 트레이딩 (시간봉)
- 전환선: 7기간
- 기준선: 22기간
- 선행스팬 B: 44기간

### 장기 투자 (주봉)
- 기본 설정 그대로 사용

## 다른 지표와의 결합

### 일목 + RSI
- 일목: 추세 방향
- RSI: 과매수/과매도 확인

### 일목 + 거래량
- 구름 돌파 시 거래량 급증 확인

### 일목 + 피보나치
- 구름과 피보나치 레벨 일치 시 강한 지지/저항

## 주의사항

- **레인징 시장에서 잘못된 신호 빈번**
- **구름은 26일 미래를 보여줌 (예측이 아님)**
- **지연 지표 (후행스팬)**
- 다른 지표와 함께 사용 필수
