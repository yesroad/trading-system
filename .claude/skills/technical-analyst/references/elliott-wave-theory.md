# Elliott Wave Theory

R.N. Elliott의 파동 이론: 시장은 반복 가능한 5-3 파동 패턴으로 움직인다.

## 기본 구조

### 충격파 (Impulse Wave): 5파동
```
      3
     /\
    /  \
   /    \     5
  /      \   /\
 /        \ /  \
1          4
```

**주 추세 방향 (1-3-5) + 조정 (2-4)**

- **파동 1**: 초기 상승 (약한 참여)
- **파동 2**: 조정 (파동 1 시작점 이탈 금지)
- **파동 3**: 강한 상승 (가장 길고 강력)
- **파동 4**: 조정 (파동 1 영역 침범 금지)
- **파동 5**: 최종 상승 (다이버전스 가능)

### 조정파 (Corrective Wave): 3파동

```
a    c
 \  /
  \/
  b
```

**A-B-C 패턴**
- A파: 초기 하락
- B파: 반등 (속임수)
- C파: 최종 하락

## 핵심 규칙 (절대 위반 불가)

1. **파동 2는 파동 1 시작점 아래로 떨어지지 않는다**
2. **파동 3는 1, 5보다 짧을 수 없다** (보통 가장 길다)
3. **파동 4는 파동 1 영역을 침범하지 않는다**

## 가이드라인 (자주 발생하나 절대적이지 않음)

- 파동 2는 보통 파동 1의 50-61.8% 되돌림
- 파동 4는 보통 파동 3의 38.2% 되돌림
- 파동 3와 5 중 하나는 파동 1의 1.618배 확장
- 파동 5는 파동 1의 길이와 유사하거나 0.618배

## 확장 (Extension)

하나의 충격파가 또 다른 5파동으로 세분화:
- 파동 3 확장 (가장 흔함)
- 파동 5 확장 (약세장 종료 신호)
- 파동 1 확장 (드물음)

## 실전 적용

### 매수 타이밍
- 파동 2 완료 후 (파동 3 시작)
- 파동 4 완료 후 (파동 5 시작)
- 조정파 C 완료 후 (새로운 충격파 시작)

### 매도 타이밍
- 파동 5 완료 예상 시
- 다이버전스 확인 (가격 신고가 but RSI/MACD 하락)

### 파동 카운트 확인
1. 피보나치 되돌림과 일치하는가?
2. 거래량 패턴이 맞는가? (파동 3 최고, 파동 5 감소)
3. 하위 시간대에서 5-3 패턴이 보이는가?

## 주의사항

- 파동 카운트는 **주관적**이다 (여러 해석 가능)
- 실시간 카운팅은 어렵다 (사후 분석은 쉬움)
- 다른 지표와 함께 사용 필수
