# Candlestick Patterns (캔들 패턴)

일본 쌀 거래상 Munehisa Homma가 개발한 차트 분석 기법

## 캔들 구조

```
      │  ← 위꼬리 (Upper Shadow)
    ┌─┐
    │ │ ← 몸통 (Real Body)
    └─┘
      │  ← 아래꼬리 (Lower Shadow)
```

- **녹색/흰색**: 양봉 (종가 > 시가)
- **빨간색/검은색**: 음봉 (종가 < 시가)
- **몸통**: 시가와 종가 차이
- **꼬리**: 장중 최고가/최저가

## 반전 패턴 (Reversal Patterns)

### 1. Hammer (망치)

```
     │
     │
   ┌─┐
   └─┘
```

**조건:**
- 하락 추세 끝
- 작은 몸통, 긴 아래꼬리 (몸통의 2배 이상)
- 위꼬리 없거나 매우 짧음

**의미:** 매도 압력 감소, 상승 반전 가능
**확인:** 다음 봉이 양봉으로 상승

### 2. Inverted Hammer (역망치)

```
   ┌─┐
   └─┘
     │
     │
```

**조건:**
- 하락 추세 끝
- 작은 몸통, 긴 위꼬리
- 아래꼬리 없거나 매우 짧음

**의미:** 매수 압력 증가
**확인:** 다음 봉 갭 상승

### 3. Shooting Star (유성)

```
   ┌─┐
   └─┘
     │
     │
```

**조건:**
- 상승 추세 끝
- 작은 몸통, 긴 위꼬리
- 아래꼬리 없음

**의미:** 상승 실패, 하락 반전
**확인:** 다음 봉이 음봉으로 하락

### 4. Hanging Man (교수형)

```
     │
     │
   ┌─┐
   └─┘
```

**조건:**
- 상승 추세 끝
- 작은 몸통, 긴 아래꼬리
- 위꼬리 없음

**의미:** 매수 세력 약화
**확인:** 다음 봉 하락

### 5. Engulfing Patterns (포용형)

#### Bullish Engulfing (강세 포용)
```
    ┌───┐
  ┌─┤   │
  │ │   │
  └─┤   │
    └───┘
  음봉 양봉
```

**조건:**
- 하락 추세 끝
- 양봉이 이전 음봉을 완전히 포용

**의미:** 강한 상승 반전

#### Bearish Engulfing (약세 포용)
```
  ┌───┐
  │   ├─┐
  │   │ │
  │   ├─┘
  └───┘
  양봉 음봉
```

**조건:**
- 상승 추세 끝
- 음봉이 이전 양봉을 완전히 포용

**의미:** 강한 하락 반전

### 6. Doji (도지)

```
    ─┼─
     │
```

**조건:**
- 시가 = 종가 (몸통 없음)
- 긴 위/아래꼬리

**의미:** 매수/매도 세력 균형, 우유부단
**종류:**
- Long-legged Doji (긴 꼬리)
- Dragonfly Doji (잠자리, 위꼬리 없음)
- Gravestone Doji (묘비, 아래꼬리 없음)

**해석:**
- 상승 추세 끝: 하락 전환 가능
- 하락 추세 끝: 상승 전환 가능

## 지속 패턴 (Continuation Patterns)

### 1. Three White Soldiers (세 병사)

```
      ┌─┐
    ┌─┤ │
  ┌─┤ │ │
  │ │ │ │
  └─┴─┴─┘
```

**조건:**
- 3개 연속 긴 양봉
- 각 봉의 시가가 이전 봉 몸통 내부
- 위꼬리 짧음

**의미:** 강한 상승 지속

### 2. Three Black Crows (세 까마귀)

```
  ┌─┬─┬─┐
  │ │ │ │
  │ │ ├─┘
  │ ├─┘
  └─┘
```

**조건:**
- 3개 연속 긴 음봉
- 각 봉의 시가가 이전 봉 몸통 내부

**의미:** 강한 하락 지속

### 3. Rising Three Methods (상승 삼법)

```
  ┌─┐       ┌─┐
  │ │ ┌┐┌┐  │ │
  │ │ └┘└┘  │ │
  └─┘       └─┘
  양봉 조정 양봉
```

**조건:**
- 긴 양봉
- 2-3개 작은 음봉 조정 (첫 양봉 범위 내)
- 마지막 긴 양봉으로 신고가

**의미:** 상승 추세 재개

## 다중 캔들 패턴

### Morning Star (샛별)

```
        ┌─┐
        │ │
  ┌─┐   │ │
  │ │ ┌┐│ │
  │ │ └┘└─┘
  └─┘
  음봉 도지 양봉
```

**조건:**
- 하락 추세 끝
- 긴 음봉 → 작은 캔들(도지) → 긴 양봉

**의미:** 강한 상승 반전

### Evening Star (저녁별)

```
  ┌─┐
  │ │
  │ │ ┌┐  ┌─┐
  │ │ └┘  │ │
  └─┘     │ │
          └─┘
  양봉 도지 음봉
```

**조건:**
- 상승 추세 끝
- 긴 양봉 → 작은 캔들 → 긴 음봉

**의미:** 강한 하락 반전

## 실전 적용

### 신뢰도 향상 조건

1. **추세 확인**: 명확한 추세 중 패턴 발생
2. **거래량**: 반전 방향으로 거래량 증가
3. **지지/저항**: 주요 레벨에서 패턴 발생
4. **다음 봉 확인**: 패턴 방향으로 다음 봉 진행

### 진입 타이밍

- **공격적**: 패턴 완성 즉시
- **보수적**: 다음 봉 확인 후

### 손절매

- 패턴의 최저점(매수) / 최고점(매도) 아래/위

### 목표가

- 패턴 크기만큼 (Measured Move)
- 다음 지지/저항까지

## 주의사항

- **단독 사용 금지**: 다른 지표와 함께 사용
- **시간대 중요**: 장기 차트일수록 신뢰도 높음
- **컨텍스트 필수**: 추세, 지지/저항 고려
- **통계적 검증**: 백테스팅 필수

## 패턴 신뢰도 순위

1. Engulfing Patterns (80%+)
2. Morning/Evening Star (75%+)
3. Three White Soldiers/Black Crows (70%+)
4. Hammer/Shooting Star (65%+)
5. Doji (60%+, 컨텍스트 의존적)
