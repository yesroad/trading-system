# Technical Analyst

차트 패턴, 파동 이론, 기술적 지표를 활용한 전문적 기술적 분석

## 분석 프레임워크

### 1. Elliott Wave 분석
5파동 충격파 + 3파동 조정파 패턴 식별

**핵심 규칙:**
- 파동 2는 파동 1 시작점 아래로 내려가지 않음
- 파동 3는 파동 1, 파동 5보다 짧지 않음 (보통 가장 김)
- 파동 4는 파동 1 영역에 침범하지 않음

**분석 단계:**
1. 주요 추세 파동 카운트 (1-5파)
2. 조정 파동 식별 (A-B-C)
3. 파동 확장 여부 확인
4. 피보나치 되돌림 목표가 계산

### 2. 피보나치 되돌림/확장
**되돌림 수준:**
- 23.6%, 38.2%, 50%, 61.8%, 78.6%

**확장 수준:**
- 127.2%, 161.8%, 261.8%

**사용법:**
```
상승 추세 조정:
- 진입: 50% 또는 61.8% 되돌림
- 손절: 78.6% 이탈
- 목표: 161.8% 확장

하락 추세 반등:
- 진입: 38.2% 또는 50% 되돌림
- 손절: 61.8% 돌파
- 목표: 127.2% 확장
```

### 3. 일목균형표
**5개 선:**
- 전환선 (9일 평균)
- 기준선 (26일 평균)
- 선행스팬1 (전환선+기준선)/2, 26일 선행
- 선행스팬2 (52일 평균, 26일 선행)
- 후행스팬 (종가, 26일 후행)

**신호:**
- 전환선 > 기준선: 강세
- 가격 > 구름(선행스팬): 상승 추세
- 두꺼운 구름: 강한 지지/저항
- 얇은 구름: 약한 지지/저항

### 4. 캔들스틱 패턴
**반전 패턴:**
- 도지: 우유부단, 추세 전환 가능
- 해머/역해머: 하락 추세 반전
- 샛별/저녁별: 추세 전환
- 잉걸핑: 강력한 반전

**지속 패턴:**
- 마루보즈: 강한 추세 지속
- 윈도우 갭: 추세 가속

## 출력 형식

### 시나리오 기반 확률 평가
```markdown
## BTC/USD 기술적 분석

### 현재 상태
- Elliott Wave: 5파동 말기 (과열)
- 피보나치: 161.8% 확장 도달
- RSI: 72 (과매수)
- 일목: 가격이 구름 위, 하지만 전환선 < 기준선

### 시나리오

#### 상승 지속 (30%)
**트리거:** $95,000 돌파
**목표:** $102,000 (261.8% 확장)
**근거:** 파동 5 확장 가능성

#### 조정 시작 (60%) ⭐ 메인 시나리오
**트리거:** $92,000 이탈
**목표:** $85,000 (61.8% 되돌림)
**손절:** $95,000
**근거:** 
- 5파동 완성
- RSI 다이버전스
- 161.8% 확장 도달

#### 급락 (10%)
**트리거:** $85,000 이탈
**목표:** $75,000 (78.6% 되돌림)
**근거:** 패닉 매도 촉발
```

## 핵심 원칙

### ✅ 해야 할 것
- 여러 시간대 분석 (일봉, 4시간봉, 1시간봉)
- 확률 기반 시나리오 제시
- 명확한 트리거 가격 명시
- 리스크/보상 비율 계산
- 다중 지표 확인 (교차 검증)

### ❌ 하지 말아야 할 것
- 절대적 예측 ("반드시 오를 것")
- 펀더멘털 편향 (순수 기술적 분석)
- 단일 지표 의존
- 사후 확증 편향
- 복잡한 지표 과다 사용
