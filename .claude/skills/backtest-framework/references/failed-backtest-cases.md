# 백테스트 실패 사례 10건

## 1. 과최적화 (Over-fitting)

**사례:** RSI 기간을 13.7일, MACD를 (11, 25, 8)로 최적화 → IS 수익률 80%, OOS -10%
**원인:** 과거 데이터에만 맞춘 파라미터
**교훈:** 파라미터는 정수 단위, Walk-Forward 필수

## 2. Look-Ahead Bias (미래 정보 참조)

**사례:** 당일 종가로 당일 매수 신호 생성
**원인:** 실제로는 종가를 알 수 없는 시점에 거래
**교훈:** 신호 생성은 전일 데이터만 사용

## 3. 생존 편향 (Survivorship Bias)

**사례:** 현재 S&P 500 구성 종목으로만 백테스트 → 수익률 +50%
**원인:** 상장폐지/퇴출 종목 미포함
**교훈:** 전체 유니버스 (상장폐지 포함) 사용

## 4. 슬리피지 미고려

**사례:** 슬리피지 0% 가정 → 백테스트 +30%, 실전 +5%
**원인:** 수수료 및 체결 가격 차이 무시
**교훈:** 브로커별 슬리피지 모델 적용

## 5. 거래량 무시

**사례:** 일일 거래량 10만원 종목 백테스트 → 실제 체결 불가
**원인:** 유동성 확인 없이 시뮬레이션
**교훈:** 최소 거래량 필터 (일일 1억원 이상)

## 6. 데이터 품질 문제

**사례:** 분할/합병 미조정 데이터 사용 → 왜곡된 수익률
**원인:** Adjusted Close 미사용
**교훈:** Yahoo Finance 등 조정 데이터 사용

## 7. 극단적 레버리지

**사례:** 매 거래마다 100% 올인 → MDD -80%
**원인:** 리스크 관리 없음
**교훈:** 켈리 기준, 최대 포지션 크기 제한

## 8. 단일 시장 의존

**사례:** 2020~2021 강세장 데이터만 백테스트
**원인:** 다양한 시장 환경 미검증
**교훈:** 최소 10년 이상, 약세장 포함

## 9. 커미션 미반영

**사례:** 고빈도 전략 (일 10회 거래) → 수수료로 수익 소멸
**원인:** 거래 비용 계산 누락
**교훈:** 모든 거래에 실제 수수료 적용

## 10. 과도한 데이터 마이닝

**사례:** 100개 지표 조합 테스트 → 우연히 좋은 결과 발견
**원인:** Multiple Testing Problem
**교훈:** 가설 먼저 세우고 검증, 지표 3~5개로 제한
