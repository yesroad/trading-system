# CANSLIM 포트폴리오 구성 규칙

## 1. 기본 원칙

**집중 투자:** 소수 정예 종목에 집중 (최대 8종목)
**빠른 손절:** -7~8% 손절 원칙
**이익 보호:** +20% 이상 수익 시 일부 차익실현

## 2. 포트폴리오 크기

```typescript
const PORTFOLIO_RULES = {
  MAX_POSITIONS: 8,              // 최대 8종목
  MIN_POSITION_SIZE: 0.10,       // 최소 10% (1종목당)
  MAX_POSITION_SIZE: 0.25,       // 최대 25% (1종목당)
  CASH_RESERVE: 0.20,            // 현금 비중 20% (약세장)
};
```

## 3. 진입 규칙

- CANSLIM Score A+ (90점 이상) 종목만 매수
- Cup-with-Handle 패턴 형성 시 진입
- 거래량 급증 (평균 대비 2배 이상) 확인
- 시장 전체가 상승 추세일 때만

## 4. 청산 규칙

### 손절
- -7% 도달 시 무조건 손절
- -3% 도달 시 재평가 (근거 약화 시 손절)

### 이익실현
- +20~25%: 50% 차익실현
- +50%: 추가 50% 차익실현
- 나머지는 Trailing Stop (최고가 대비 -8%)
